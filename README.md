# React Admin Dashboard

A clean admin dashboard UI built with React, TypeScript, Vite, SCSS, Material UI Data Grid, React Query, and Recharts.

## Features

- Dashboard overview with cards and charts
- Users and products tables
- User and product detail pages
- Sidebar and top navigation
- Modal form UI for adding records

## Tech Stack

- React
- TypeScript
- Vite
- React Router
- React Query
- MUI Data Grid
- Recharts
- Sass

## Routes

- `/`
- `/users`
- `/users/:id`
- `/products`
- `/products/:id`
- `/login`

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```
